Snake-Water-Gun Game

Snake-Water-Gun is a simple command-line game in Python where you play against the computer. It's inspired by the classic rock-paper-scissors game, but with Snake, Water, and Gun.


 How to Play

1. Run the Python script.
2. Choose one option from: Snake, Water, Gun.
3. The computer will randomly select an option.
4. Rules:
   - Snake vs Water → Snake wins
   - Water vs Gun → Water wins
   - Gun vs Snake → Gun wins
5. Each game has 5 rounds.
6. At the end, the winner is announced based on total points.
